单位简介
单位简介:

上海市体育局主要职责内设机构和人员编制规定根据《中共中央办公厅、国务院办公厅关于印发〈上海市人民政府机构 改革方案〉的通知》的规定，设立上海市体育局，为市政府直属机构。上海市体育局设8个职能处室， 即办公室（外事处、法规处），党委办公室、群众体育处、竞技体育处、竞赛处、科教处、人事处、计划财务处。按有关规定 设置纪检监察机构。截至2008年12月31日，上海市体育系统从业人员总计6134人，其中公务员243人、教练员1119人、运动员 1016人、科研人员79人、医务人员70人、文化教师307人、管理人员1427人、工勤人员1320人，其他553人。
